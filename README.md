# Frontend Angular synchronisé avec le backend Flight

Le frontend consomme maintenant directement :
- POST /api/v1/Account/login
- POST /api/v1/Account/register
- POST /api/v1/Account/refresh-token
- POST /api/v1/Account/forgot-password
- POST /api/v1/Account/reset-password
- POST /api/v1/Account/confirm-email
- GET /api/v1/Account/user
- GET /api/v1/Account/profile
- GET /api/v1/Account/activity
